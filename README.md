> The project name and logo is source from [Dung beetle](https://en.wikipedia.org/wiki/Dung_beetle#Scarab_in_ancient_Egypt)

## Deploy

Packed all dependenices into Docker, easy peasy:

```
$ docker-compose up -d
```

That's all!
